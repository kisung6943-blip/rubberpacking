# PRD — 압력밥솥 고무패킹 실수하지 않고 주문하는 방법 (앱)

**문서 버전:** v1.0  
**작성일:** 2026-04-07  
**작성자:** 이에스리빙  
**상태:** 초안

---

## 1. 개요 (Overview)

### 1.1 배경

이에스리빙 쇼핑몰에서 휘슬러·WMF·타파웨어·실리트 압력밥솥용 고무패킹을 판매 중이다. 고객의 오주문이 잦아 반품·교환 비용과 CS 응대 부담이 크다.

**주요 오주문 원인:**
- 브랜드 구분 없이 지름만 재고 아무 패킹이나 주문
- 내경/외경 구분 없이 측정 (고무 두께 포함)
- 같은 브랜드라도 모델별 패킹 호환 불가 (특히 휘슬러 구형/신형)
- 같은 용량이라도 모델에 따라 사이즈 다름 (WMF 3L 익스프레스 vs 퍼펙트)

### 1.2 목적

고객이 브랜드 → 모델 → 용량 순서로 선택하면 자신의 압력솥에 맞는 정확한 고무패킹 규격을 안내받을 수 있는 웹 기반 인터랙티브 가이드 앱을 제공한다.

### 1.3 목표

- 오주문율 감소
- CS 문의 감소
- 고객의 올바른 구매 유도 및 재구매율 향상

## 2. 사용자 (Target Users)

| 구분 | 설명 |
|------|------|
| 주 사용자 | 압력밥솥 고무패킹 교체가 필요한 고객 |
| 연령대 | 30~60대 (디지털 기기 사용에 익숙하지 않을 수 있음) |
| 사용 환경 | 모바일 브라우저 (쇼핑몰 상세페이지 or 링크 공유) |
| 브랜드 | 휘슬러(Fissler), WMF, 타파웨어, 실리트(Silit) 보유 고객 |

---

## 3. 사용자 플로우 (User Flow)

```
앱 진입
  ↓
[STEP 1] 브랜드 선택
  ├── 휘슬러 → [STEP 2] 모델 선택 → [STEP 3] 용량 선택 → 결과
  ├── WMF · 타파웨어 → [STEP 2] 용량 선택 → 결과
  ├── 실리트 → [STEP 2] 용량 선택 → 결과
  └── 타브랜드 압력솥 → 안내 메시지 (구매 불가)
```

## 4. 기능 명세 (Feature Specification)

### 4.1 헤더 영역

| 항목 | 내용 |
|------|------|
| 제목 | 압력밥솥 고무패킹 실수하지 않고 주문하는 방법 |
| 부제 | 브랜드 → 모델 → 사이즈 순서로 선택하면 정확한 패킹을 찾아드려요 |
| 경고 문구 | 지름만 재시고 아무 고무패킹이나 주문하시면 사용이 안됩니다 (빨간색 강조) |

---

### 4.2 STEP 1 — 브랜드 선택

2×2 그리드 버튼 구성

| 버튼 | 부제 | 동작 |
|------|------|------|
| 휘슬러 | Fissler | → STEP 2 모델 선택으로 이동 |
| WMF · 타파웨어 | 단일 모델 | → STEP 2 용량 선택으로 이동 |
| 실리트 | Silit · 단일 모델 | → STEP 2 용량 선택으로 이동 |
| 타브랜드 압력솥 | 브랜드 모름 · 구매 불가 | → 안내 메시지 표시 (점선 버튼) |

**타브랜드 선택 시 안내 메시지:**
> 휘슬러 · WMF · 타파웨어 · 실리트 이외의 브랜드는 현재 취급하지 않아요.

---

### 4.3 STEP 2 — 모델 선택 (휘슬러 전용)

WMF · 타파웨어 · 실리트는 단일 모델이므로 이 단계 생략, 바로 용량 선택으로 이동

| 버튼 위치 | 모델명 | 태그 색상 |
|-----------|--------|-----------|
| 좌측 | 구형 (A모델) | 주황 계열 |
| 우측 | 신형 (B모델) | 초록 계열 |

**뚜껑캡(압력추) 선택 (휘슬러 전용, 모델 선택 후 표시):**

| 버튼 | 부제 |
|------|------|
| 소형캡 | 18cm용 |
| 중형캡 | 22cm용 |
| 대형캡 | 26cm용 |

> 패킹과 함께 교체 권장 안내 문구 포함

---

### 4.4 STEP 3 — 용량 선택

선택 전 내경/외경 측정 가이드 박스 표시 (SVG 일러스트 포함)

| 항목 | 설명 |
|------|------|
| 내경 (안쪽) | 올바른 측정 — 고무 두께 제외, 안쪽만 측정 |
| 외경 (바깥) | 잘못된 측정 — 고무 두께 포함됨, 오주문 원인 |

**브랜드별 용량 및 내경 데이터:**

#### 휘슬러 (구형 A모델 / 신형 B모델 공통)

| 용량 | 내경 |
|------|------|
| 2.5L | 18cm (180mm) |
| 4.5L | 22cm (220mm) |
| 6L | 26cm (260mm) |

> 구형(A모델) 선택 시 경고: "구형(A모델) 패킹은 신형(B모델)과 호환되지 않습니다. 모델을 꼭 확인 후 구매하세요."

#### WMF · 타파웨어

| 용량 | 내경 | 비고 |
|------|------|------|
| 2.5L | 18cm (180mm) | 전 모델 |
| 3L (익스프레스) | 20cm (200mm) | 퍼펙트/플러스와 다름 주의 |
| 3L (퍼펙트/플러스) | 22cm (220mm) | 익스프레스와 다름 주의 |

#### 실리트 (Silit)

| 용량 | 내경 |
|------|------|
| 2.5L | 18cm (180mm) |
| 3.0L | 22cm (220mm) |

---

### 4.5 결과 화면

용량 선택 시 자동으로 결과 카드 표시

| 항목 | 내용 |
|------|------|
| 브랜드 배지 | 선택한 브랜드명 (파란 배경) |
| 패킹명 | "[모델명] [용량] 전용 패킹 (+ 뚜껑캡 선택 시 표시)" |
| 설명 문구 | "아래 규격의 패킹을 구매하시면 정확히 맞습니다." |
| 규격 정보 | 내경, 해당 용량, 모델명 (테이블 형식) |
| 뚜껑캡 | 선택한 경우 추가 표시 |
| 경고 박스 | 호환 불가 등 주의사항 (해당 시만 표시, 주황 배경) |
| 주문 버튼 | "이 패킹 주문하기 ↗" (파란 버튼) |
| 다시 선택 | "다시 선택하기" (보조 버튼) |

## 5. UX / UI 요구사항

| 항목 | 요구사항 |
|------|----------|
